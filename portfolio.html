PHP Email & Template System (ACW 3: Cutting Edge)

In short I have added an extension to my contact form. It allows the user to insert their data. Once they submit the form, PHP will build a dynamic email template that looks pretty.

My task was to make use of script/programming (including database) - JavaScript, jQuery, Ruby or PHP. For my task I chose to develop and extension for my website written in PHP because this is where my main skills are. I have developed a PHP email template system using static methods to make using my code a little bit more intuitive. The method calls look like this: Email::to("john.crossley@stu.mmu.ac.uk");. From this example you can see we are telling the email object to set the to email to john.crossley@stu.mmu.ac.uk. We set the other data in a similar way.

The system works by allowing a designer to code up a standard HTML email page and style it using CSS as normal. Whenever the designer needs to include unique data that they will provide from the contact form they simply insert placeholders like so <h1>Welcome, {{full_name}}</h1>. After they have designed the template they store it in a folder called email_templates.

When the user fills out their information in the contact form it will submit the data to PHP and then PHP will parse the file and convert all the placeholders with the correct user data, and email the user.
