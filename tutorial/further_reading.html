Building a Blog with Rails

Further Reading.

Well you have reached the end of my tutorial, my name is John Crossley and I hope you enjoyed learning a little bit of Ruby on Rails. I certainly enjoyed teaching you guys.

Ruby is growing fast and becoming more better each time a new version is released. Below I have provided you with some further reading material to help you master the Rails framework and Ruby language.

Learning Ruby

  • Any new rubyist's first port of call, learn the Ruby programming language by using the interactive Ruby tutorial. http://tryruby.org/levels/1/challenges/0

  • Help kickstart your Ruby career by looking at the Ruby quickstart guide.
    http://www.ruby-lang.org/en/documentation/quickstart/

  • All Ruby programmers should have this page bookmarked, refer to the documentation to help you use the Ruby language. http://ruby-doc.org/core-1.9.3/

Learning Rails

  • A paid tutorial for learning Rails, check out this link to grasp a solid introduction to building Ruby applications like the professional. http://ruby.railstutorial.org/

  • A great resource for learning Rails, here is a collection of tutorials to help you develop your rails knowledge. http://net.tutsplus.com/tutorials/ruby/the-best-way-to-learn-ruby-on-rails/

  • A collection of Rails screecasts to help you learn different types of Rails programming. http://rubyonrails.org/screencasts
